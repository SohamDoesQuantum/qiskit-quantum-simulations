# Quantum Algorithms: Theoretical Study and Qiskit Simulations


---

## Abstract

This repository contains the Qiskit-based implementation and simulation of several foundational quantum algorithms studied as part of a project in quantum computation. The primary objective is to complement the mathematical formulation and theoretical analysis of these algorithms with practical quantum circuit realizations and simulations.

The notebook demonstrates the construction of quantum circuits, oracle implementations, simulation using Qiskit Aer, and verification of theoretical predictions through measurement outcomes and visualizations.

---

## Algorithms Covered

- Deutsch Algorithm
- Deutsch–Jozsa Algorithm
- Simon's Algorithm
- Grover's Algorithm
- Shor's Algorithm

---

## Objectives

- Study the theoretical foundations of fundamental quantum algorithms.
- Construct the corresponding quantum circuits using Qiskit.
- Simulate algorithm execution on quantum simulators.
- Verify expected outcomes through measurement statistics.
- Analyze the computational advantages offered by quantum computation over classical methods.

---

## Software Stack

- Python
- Qiskit
- Qiskit Aer
- Matplotlib
- Google Colab

---

## Contents of the Notebook

Each algorithm section includes:

1. Brief theoretical background and motivation.
2. Quantum circuit construction.
3. Oracle implementation.
4. Simulation using Qiskit Aer.
5. Circuit visualization.
6. Measurement results and histograms.
7. Discussion and interpretation of results.

---

## Quantum Advantage Summary

| Algorithm | Problem Addressed | Classical Complexity | Quantum Complexity |
|------------|------------------|---------------------|-------------------|
| Deutsch | Constant vs Balanced Function | O(2) | O(1) |
| Deutsch–Jozsa | Generalized Constant vs Balanced Problem | O(2ⁿ) | O(1) |
| Simon | Hidden XOR String Problem | Exponential | Polynomial |
| Grover | Unstructured Search | O(N) | O(√N) |
| Shor | Integer Factorization | Sub-exponential | Polynomial |
